Principe d'une flashcard

Une flashcard est constituée de deux faces :

  • le recto avec une question
  • le verso avec la réponse

Quelle est l'expression vectorielle du poids ?

\(\vec{P} = m \times \vec{g}\)

Méthode Leitner

  • 5 boîtes (A, B, C, D, E) pouvant contenir les flashcard :
    • A : contient toutes les cartes au début
  • Tous les jours répondre aux questions de la boîte A :
    • Si réponse correcte, la flashcard passe dans la boîte B
    • Sinon elle reste dans la boîte A
  • Tous les 3 jours rajouter les cartes de la boîte B :
    • Si réponse correcte, la carte passe dans la boîte C
    • Sinon elle repasse dans la A
  • Toutes les semaines, rajouter les cartes de la boîte C
  • Toutes les 2 semaines, rajouter les cartes de la boîte D

  • Tous les mois, rajouter les cartes de la boîte E
